제어문

프로그램의 실행 순서(=흐름)를 제어하는 문장(=문법, 구문)이다.

제어문은 크게 1)조건문, 2)선택문, 3)반복문으로 구분한다.

조건문

조건식의 값이 true, false에 따라 각각 다른 코드를 실행하는 문법이다.

특정 조건에 의해서 프로그램 실행이 분기(=나누어지는 것)

  1. 단순 if
  2. if ~ else
  3. if ~ else if
  4. 중첩 if

단순 if

조건식이 true일때만 실행되는 구문, false일때는? 실행할 게 없음.

            if(조건식) {
                ...자바스크립트 코드들...;
            }
