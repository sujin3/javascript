if~ else if

두가지 이상의 조건식과 정해놓은 조건을 만족하지 않았을 때 실행되는 코드로 이뤄져 있습니다. 

            if(조건식1) {
                조건식1이 true면 실행될 코드1;
            } else if(조건식2) {
                조건식2가 true면 실행될 코드2;
            } else if(조건식3,4,5,...) {
                조건식3,4,5가 true면 실행될 코드 3,4,5;
            } else {
                조건식 1~5까지 일치하는 게 없으면
            }

if ~ else if 예시

질의응답창을 이용해서, 현재는 몇 월인지 묻고 else if문을 이용해서 현재 월에 해당하는 계절과 관련된 문구를 출력하라
