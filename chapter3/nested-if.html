중첩 if

if 구문 안에 또 if구문을 작성하는 규칙, 너무 많이 중첩(=nested) 하게 될 경우 복잡한 코드로 오류나 실수를 범할 수 있음 

            if(조건식1) {
                if(다른 조건식2) {
                 코드
                }
            }

중첩 if 예시

질의 응답창을 이용해 로그인(id/pw) 하는 프로그램을 작성할 것

미리 정해진 id/pw 와 사용자가 입력한 id/pw를 비교해서, id가 일치하면, pw를 비교해서 그것마저 일치했을 때 로그인 성공하고, id가 일치하나, pw를 비교해서 일치하지 않다면 로그인 실패 또는 id마저 미리 정해진 id와 다르다면, 그냥 실패!(pw가 맞더라도)

로그인 프로그램은 사용자가 입력한 정보와 회원정보의 일치유무에 따라 성공, 실패에 따른 화면을 제공하는 것
